# Halalgo line自動回覆機器人
台灣有許多的穆斯林朋友們，但對於穆斯林友善的環境或設備相當有限。
伊斯蘭教有『清真』的飲食規範，但在台灣，清真餐廳數量有限，穆斯林朋友們在尋找餐廳時非常不方便，且有時候可能面臨『破戒』問題。於是，為了更加尊重他們的信仰，以及協助打造穆斯林友善環境，架設了這個HalalGo機器人，希望能讓穆斯林朋友們在尋找清真餐廳上更加方便。目前尚未開發英文版，正持續努力中...
